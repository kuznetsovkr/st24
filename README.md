# Her Shop - заготовка интернет-магазина

Базовый каркас фронтенда и бэкенда под MVP интернет-магазина (React + Node/Express + PostgreSQL).

## Что уже есть
- Frontend (Vite + React + TS) с маршрутами под карту сайта: `/`, `/catalog`, `/catalog/:slug`, `/cart`, `/about`, `/b2b`, `/privacy`, `/terms`, `/admin` и 404.
- Модалка быстрой карточки товара.
- Админка с добавлением/редактированием/удалением товаров (название, SKU, цена, описание, изображения).
- Backend (Express + TS) с API для товаров/категорий, авторизацией по телефону и хранением в PostgreSQL.
- Загрузка изображений в папку `server/uploads` и раздача через `/uploads/*`.

## Локальный запуск
Установите зависимости в воркспейсах:
```bash
npm install
```

Запуск фронтенда:
```bash
npm run dev --workspace client
```

Запуск бэкенда:
```bash
npm run dev --workspace server
```

Сборка бэкенда:
```bash
npm run build --workspace server
```

## Переменные окружения
Скопируйте `server/.env.example` в `server/.env` и укажите:
- `DATABASE_URL` - строка подключения к PostgreSQL.
- `JWT_SECRET` - секрет для подписи токенов.
- `ADMIN_PHONE` - номер администратора (по умолчанию `79964292550`).

Создайте базу данных и укажите ее в `DATABASE_URL` (например, `createdb her`).
После запуска сервер сам создаст таблицы и заполнит категории.
